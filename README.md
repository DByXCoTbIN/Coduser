# Brauser IDE

Редактор кода со встроенным браузером и серверами для веб-разработки.

## Что умеет

- **Live preview** — страница обновляется при каждом сохранении файла
- **Редактор** — Monaco Editor с подсветкой PHP, JS, HTML, CSS и других языков
- **Серверы** — встроенные PHP, Apache, MySQL с управлением через интерфейс
- **PHPMyAdmin** — работа с базой данных прямо в окне программы
- **Терминал** — выполнение команд без выхода из редактора
- **Git** — коммиты, ветки, история изменений
- **Кроссплатформенность** — работает на Linux, macOS, Windows

## Запуск

```bash
# Установить зависимости
npm install

# Запустить приложение
electron . --no-sandbox --disable-gpu --disable-dev-shm-usage
```

## Горячие клавиши

| Действие | Комбинация |
|----------|------------|
| Открыть папку | Ctrl+Shift+O |
| Сохранить файл | Ctrl+S |
| Новый файл | Ctrl+N |
| Терминал | Ctrl+` |
| Обновить preview | F5 |
| Меню | Alt |

## Управление серверами

В боковой панели **Servers**:

- **PHP Server** — стартует `php -S` для текущего проекта
- **MySQL** — управление через systemctl
- **Apache** — управление через systemctl
- **PHPMyAdmin** — открывается в окне preview

## Структура

```
brauser-ide/
├── electron/       # Основной процесс Electron
├── ide/            # Интерфейс (HTML, CSS, JS)
├── servers/        # Конфиги и файлы серверов
│   ├── www/        # Документ-корень по умолчанию
│   └── phpmyadmin/ # PHPMyAdmin
└── package.json
```

## Технологии

- Electron — каркас приложения
- Monaco Editor — редактор кода
- PHP — встроенный dev-сервер
- FastAPI — Python backend для операций с файлами и серверами
